Optimum Role Card

Đăng nhập Discord, lấy role từ server Optimum và tải card kỷ niệm.

Chưa kết nối.

Cấu hình nhanh trước khi deploy
  1. Tạo Discord OAuth2 app và lấy Client ID.
  2. Trong Discord Developer Portal, thêm Redirect URL đúng domain GitHub Pages của bạn.
  3. Sửa CLIENT_ID, GUILD_ID và các ROLE_IDS trong app.js.
  4. Deploy thẳng repo này lên GitHub Pages.
